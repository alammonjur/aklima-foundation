# Foundation Documents

Add important foundation documents:

## Examples:
- Annual reports and financial statements
- Registration certificates
- Legal documents and bylaws
- Program brochures and flyers
- Impact assessment reports
- Partnership agreements

## Naming Convention:
- `annual-report-2024.pdf`
- `registration-certificate.pdf`
- `education-program-brochure.pdf`
- `impact-report-2024.pdf`

## Recommended Format:
- PDF format for documents
- Optimize for web (compress large files)
- Ensure documents are searchable
- Include version dates in filenames
